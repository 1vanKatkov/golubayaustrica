# Django poker site

Минимальный Django проект с лендингом покерного клуба. Страница описывает предложения клуба и выдаёт живую ленту лучших игроков.

## Структура

- `manage.py` – CLI для запуска сервера и миграций.
- `poker_site/` – конфигурация проекта (настройки, URL, WSGI/ASGI).
- `home/` – приложение с моделью `Player`, шаблоном и административной панелью.
- `home/static/photo/` – сюда помещайте файлы фотографий игроков (имена файлов хранятся в `photo_filename`).
- `home/migrations/0001_initial.py` – миграция для модели игроков.
- `requirements.txt` – зависимости проекта.

## Как запустить

1. Создайте виртуальное окружение и активируйте его (например, `python -m venv venv` + `.\venv\Scripts\Activate.ps1`).
2. Установите зависимости: `python -m pip install -r requirements.txt`.
3. Примените миграции: `python manage.py migrate`.
4. Создайте игроков через `python manage.py createsuperuser` и административную панель (`/admin/`), указывая имя, описание и имя файла фото.
5. Копируйте изображения игроков в `home/static/photo/`, имя файла должно совпадать с тем, что хранится в `Player.photo_filename`.
6. Запустите сервер: `python manage.py runserver` и откройте `http://127.0.0.1:8000/`.
